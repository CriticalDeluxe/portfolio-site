# Portfolio Site (Astro)

Minimal Astro + TypeScript scaffold for a personal portfolio.

## Requirements

- Node.js 18+ (Node.js 20+ recommended)
- npm

## Run locally

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal (typically `http://localhost:4321`).

## Included pages

- `/` landing page placeholder
- `/skills` skills index placeholder

## Project structure

- `src/layouts/BaseLayout.astro` shared layout with header/footer
- `src/pages/index.astro` landing page
- `src/pages/skills/index.astro` skills page
